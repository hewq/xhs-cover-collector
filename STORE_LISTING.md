# Chrome Web Store 商品详情

## 插件名称

小红书封面灵感夹

## 简短描述

收藏、整理、拆解和导出小红书封面与标题灵感。

## 详细描述

小红书封面灵感夹是一款为小红书创作者、内容运营、知识付费博主和自媒体从业者设计的灵感收藏工具。

当你在浏览小红书或其他网页时，可以一键保存当前页面标题、链接、作者、封面图和备注，并按照分类和标题类型整理灵感。

插件支持：

- 收藏当前页面标题和链接
- 尝试自动识别小红书页面标题、作者和封面图
- 手动编辑标题、作者和封面图链接
- 添加分类、标题类型和备注
- 搜索灵感库
- 按分类筛选
- 一键复制灵感拆解模板
- 根据标题类型生成改写标题候选
- 导出 CSV
- 导出 Markdown
- 自动保存默认领域和分类偏好

适合用于：

- 小红书选题库
- 爆款标题收集
- 封面设计参考
- 内容复盘
- 账号对标分析
- 课程和咨询内容选题

当前版本数据默认保存在浏览器本地，不上传服务器。

生产力工具 / 写作工具 / 社交媒体工具
